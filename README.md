Asteroid-Destroyer
==================

My first Ruby game. It is written in ruby using Gosu.

Controls:
----------

* Arrows - moving the ship around
* Space - shooting

Contents:
-----------

* Main.rb - main script
* media - all of the images
* supplementary - icon in different formats
